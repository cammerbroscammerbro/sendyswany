Quantum-safe Encryption API

Secure your data with post-quantum cryptography. Built for developers who need forward-secure protection.

Authentication

All requests require an API key in the header. Manage keys in the CipherQ Dashboard.

X-API-KEY: your_api_key_here

Base URL

https://api.cipherq.fronti.tech

POST /encrypt

Encrypt plaintext using quantum-resistant algorithms.

POST /encrypt

Body

{
  "message": "Hello Quantum!"
}

Response

{
  "ciphertext": "k29dkad923..",
  "key": "QSAFE-8392X...",
  "algorithm": "QuantumHybrid256"
}

POST /decrypt

Decrypt ciphertext with a provided key.

POST /decrypt

Body

{
  "ciphertext": "k29dkad923..",
  "key": "QSAFE-8392X..."
}

Usage Examples

import requests

url = "https://api.cipherq.fronti.tech/encrypt"
headers = {"Content-Type": "application/json", "X-API-KEY": "your_api_key"}
data = {"message": "hello quantum"}
res = requests.post(url, headers=headers, json=data)
print(res.json())
import fetch from "node-fetch";

const res = await fetch("https://api.cipherq.fronti.tech/encrypt", {
  method: "POST",
  headers: {"Content-Type": "application/json", "X-API-KEY": "your_api_key"},
  body: JSON.stringify({ message: "hello quantum" })
});

console.log(await res.json());
curl -X POST https://api.cipherq.fronti.tech/encrypt \
-H "Content-Type: application/json" \
-H "X-API-KEY: your_api_key" \
-d '{"message": "hello quantum"}'
